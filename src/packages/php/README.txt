          requirements
1.- have the php decompressed. 
2.- make installed
3.- the instance must be linked with devpanel. 

          Sintax of the script 
compile.sh [version with use like: 5.4.9 or 5.5.5] [path of the php decompressed without the last slash example: /home/php-5-5-5 ]

           Notes 
the script will compile the php on this path /opt/webenabled/bin/packages/php/$version and all the binaries of php will be there.
